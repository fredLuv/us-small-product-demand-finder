# 落地页与广告验证 Brief

## 英文落地页结构

1. Hero
   - Product name
   - One-sentence promise
   - Hero image or short demo clip
   - Price anchor
   - CTA: Add to Cart / Join Waitlist / Reserve for $1

2. Problem
   - 2-3 bullets describing the everyday pain.
   - Use plain consumer language, not supplier language.

3. Demo
   - Show before / during / after.
   - Keep claim observable: faster, cleaner, easier, brighter, more organized.

4. Why it works
   - 3 bullets tied to product features.
   - Avoid medical, body transformation, or guaranteed outcome language unless legally cleared.

5. FAQ
   - Shipping estimate
   - Return policy
   - What is included
   - Compatibility / sizing / care instructions

6. CTA repeat
   - Same CTA as hero.
   - Optional urgency: limited test batch, early access, launch discount.

## 三条广告素材模板

### 1. 痛点型

Hook:
> Still dealing with [annoying situation] every day?

Body:
Show the messy/frustrating moment, then introduce the product as the simple fix.

CTA:
> See if the test batch is still open.

### 2. 演示型

Hook:
> I did not expect this tiny thing to make [task] this easy.

Body:
Show the product in use with a clear before/after.

CTA:
> Try it before the first batch closes.

### 3. 对比型

Hook:
> I used to do this the hard way.

Body:
Split-screen: old way vs. product way.

CTA:
> Join the early access list.

## 平台风险语言清单

避免：
- Cure, treat, heal, guaranteed results
- Medical diagnosis or disease claims
- Before/after body claims that imply unrealistic transformation
- Trademarked brands, characters, logos, lookalike claims
- “FDA approved” unless documented and legally reviewed

优先：
- Simple, easier, more organized, compact, travel-friendly
- Helps reduce clutter
- Designed for daily use
- Test batch, early access, launch offer
